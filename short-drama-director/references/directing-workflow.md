# Director Workflow

Use this reference for beat analysis, elastic shot splitting, shot groups, and scene transitions.

## Beat map

Extract source-only beats: action, dialogue, reaction, prop, space, emotion, and time. Do not treat punctuation as a cut list. A single line may need several visual covers; several actions may remain in one carrying shot when they serve one result.

Classify information inside each beat:

- Primary: the one viewing task the audience must understand and remember.
- Secondary: relationship, space, cause, or prop information that supports the primary task.
- Background: state needed for realism or continuity but not active attention.

One primary task does not mean one character or one verb. Several people and sequential actions may remain together when they converge on the same result.

## Director decision priority

When rules compete, protect plot and motive first, then primary information, spatial/relationship continuity, performance progression, rhythm/cut value, visual expression, and finally generation stability. Do not make a higher-priority decision worse merely to simplify generation.

## Primary-information test

For each proposed shot, answer:

1. What must the audience notice first?
2. What must remain understood when the shot ends?
3. Can the composition show the subject, action target, and necessary prop?
4. Does a cut provide something the current view cannot?

Split when the primary subject, information readability, emotional phase, action phase, space, direction, prop state, sound interpretation, or bodily resource conflict changes. Combine when time, space, goal, viewing priority, axis, and result remain unified.

## Complex-scene decision order

Before splitting a complex beat, decide in this order:

1. Can one stable composition carry the primary task?
2. Can foreground/middle/background staging, blocking, sound order, or focus transfer hand attention between established subjects?
3. Can one carrying shot plus one necessary insert reveal the unreadable information?
4. Only then split into real shots if core actions compete, a critical reaction/result cannot be read, or body/prop/dialogue resources conflict.

Lower secondary/background motion rather than deleting established people. Internal focus transfer needs a causal trigger, the first task must finish or reduce, and the next subject must already have a known position.

## Carrying shot before inserts

Design one carrying shot that establishes subject, target, and spatial relation. Add an insert only to reveal necessary information, show an unreadable contact/result, compress a nonessential process, hide unstable action, or redirect attention for a motivated reveal. An insert must change audience knowledge.

## Shot groups

- Observation: notice -> target -> reaction/action.
- Action: prepare -> execute/contact -> result/consequence.
- Dialogue: speaker -> listener -> relationship/action change.
- Suspense: anomaly -> attention -> delayed reveal -> answer.
- Space: enter -> eyeline/path -> reveal -> position established.

Fast action may use more cuts, but periodically re-establish positions, exits, obstacles, and threat direction.

## Duration and action load

| Type | Typical duration |
|---|---:|
| Insert | 0.8-2s |
| Ordinary narrative | 2-4s |
| Carrying shot | 4-6s |
| Emotional hold / continuous blocking | 6-8s |
| Long take | Above 8s only with continuous visible change |

Check each character's hands, body motion, attention, gaze, mouth/dialogue, and held props. Split or simplify if the same resource must perform incompatible tasks. Do not count people or actions as an automatic split trigger.

Complete a dialogue budget before finalizing the duration table. Count audible Chinese characters, select the delivery range from `dialogue-action-sound.md`, add source-required pauses, and add time for actions that cannot overlap intelligibly with speech. Recheck the combined duration when one line continues across several covers.

## Scene-transition motive

Before a new scene, establish one source-supported motive: decision, movement, eyeline, sound, prop, line result, visible time-state change, or emotional consequence. Hard cuts and contrast cuts are valid when purposeful. Do not invent a bridge that erases an intentional rupture.

Do not decorate every boundary. Run the selection test in `transition-design.md` and record only transitions whose exit anchor, entry anchor, and narrative purpose are all explicit.

## Director audit

For every neighboring pair, explain internally:

`previous shot leaves a viewing need -> current shot answers/delays it -> current shot creates the next need`

If this sentence cannot be completed, merge, delete, or redesign the cut.
