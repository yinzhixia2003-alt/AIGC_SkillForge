# Translation Workflow

## Lock the shot

Before writing, lock the director-authoritative fields: number, duration, scene, visible people, action, shot size, angle, composition, camera movement, sound, and exact dialogue. A translation cannot change these.

## Internal shot card

Determine internally:

```text
shot task:
primary information:
secondary information:
background continuity:
relationship to prior shot:
start state:
action process:
end state:
next viewing motive:
```

## Start state

Use one of:

1. the prior end state for a continuous relationship;
2. the approved state that survives a state-only cut;
3. the first setup of a new scene;
4. storyboard A frame.

State posture/movement, depth and screen position, facing/eyeline, hands and props, doors/vehicles/furniture/damage, camera position, and active off-screen pressure. Do not say only `保持上一镜`.

## Action process

Write `first -> change/process -> final`. Do not repeat an action start or add a new outcome. A short shot gets one main action. A long shot may contain continuous blocking when the path and order are explicit.

## End state

Make position, posture, gaze, prop state, environment state, camera stop, and remaining off-screen activity concrete enough for the next continuous shot.

## Camera behavior

Inherit the director's camera choice. Describe start, direction, followed/revealed subject, speed character, and stop point. A fixed shot stays fixed. A follow shot keeps its relative side and distance. A gaze reveal follows the established eyeline.

## Sound

Use only approved dialogue, phone, OS/VO/off-screen sound, environment, action sound, bridge, or music. Describe audible facts objectively. Keep dialogue characters and punctuation in order.

## Relationship classification

- Continuous: carry concrete frame/action state and allow motivated matches/bridges.
- State-only: carry only persistent state; permit visual hard cut.
- Information-only: preserve answer/contrast but no physical continuity.
- Intentional hard cut/new scene: explicitly avoid prior physical elements.
- Unclear: remain conservative and return if a choice changes intent.
